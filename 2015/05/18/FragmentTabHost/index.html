FragmentTabHost

  • Android Widget
  • FragmentTabHost是TabHost的子类，是特殊的TabHost专门处理Tab中的Fragment。官方的解析如下：

Special TabHost that allows the use of Fragment objects for its tab content. When placing this in a view hierarchy, after inflating the hierarchy you must call setup(Context, FragmentManager, int) to complete the initialization of the tab host.

  • 使用示例 Java
  • 使用示例 XML
  • FragmentTabHost和TabHost的区别：
    • 在TabHost内部的Tab的切换是通过View.VISIBLE、View.GONE来实现的
    • 在TabHost的setCurrentTab(int index)代码中是这样实现的
  • 在FragmentTabHost中是这样切换的,FragmentTabHost实现了TabHost.OnTabChangeListener
  • 可以看到在FragmentTabHost中摘要用FragmentTransaction来attach和detach Fragment。
