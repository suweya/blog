retrofit

Retrofit Usage

使用步骤

  • 定义REST API接口

      public interface GitHubService {
    @GET("/users/{user}/repos")
    List<Repo> listRepos(@Path("user") String user);
  }
  • @Get —- 定义请求方式，也可以是POST
  • @Path(“user”) —- 对应URL(/users/{user}/repos)中的{user}
  • URL中的参数也可以这样添加
    @GET("/group/{id}/users")
    List<User> groupList(@Path("id") int groupId, @Query("sort") String sort);

  • 复杂的参数可以通过以下格式添加@QueryMap

      @GET("/group/{id}/users")
  List<User> groupList(@Path("id") int groupId, @QueryMap Map<String, String> options);

  • Post数据的添加@Body

      @POST("/users/new")
  void createUser(@Body User user, Callback<User> cb);

  • 表单数据@FormUrlEncoded @Field

      @FormUrlEncoded
  @POST("/user/edit")
  User updateUser(@Field("first_name") String first, @Field("last_name") String last);

  • 多媒体数据@Multipart @Part

      @Multipart
  @PUT("/user/photo")
  User updateUser(@Part("photo") TypedFile photo, @Part("description") TypedString description);

  • HTTP头添加

      @Headers("Cache-Control: max-age=640000")
  @GET("/widget/list")
  List<Widget> widgetList();

  @Headers({
      "Accept: application/vnd.github.v3.full+json",
      "User-Agent: Retrofit-Sample-App"
  })
  @GET("/users/{username}")
  User getUser(@Path("username") String username);

  @GET("/user")
  void getUser(@Header("Authorization") String authorization, Callback<User> callback)

  • 通过请求拦截器为每个请求添加头信息

      RequestInterceptor requestInterceptor = new RequestInterceptor() {
    @Override
    public void intercept(RequestFacade request) {
      request.addHeader("User-Agent", "Retrofit-Sample-App");
    }
  };

  RestAdapter restAdapter = new RestAdapter.Builder()
    .setEndpoint("https://api.github.com")
    .setRequestInterceptor(requestInterceptor)
    .build();

  • 通过RestAdapter来生成GitHubServices的实现类

      RestAdapter restAdapter = new RestAdapter.Builder()
      .setEndpoint("https://api.github.com")
      .build();

  GitHubService service = restAdapter.create(GitHubService.class);

  • 调用REST API

      List<Repo> repos = service.listRepos("octocat");

  • API调用在同步或者异步

    • 同步模式

        @GET("/user/{id}/photo")
  Photo getUserPhoto(@Path("id") int id);

    • 异步模式

        @GET("/user/{id}/photo")
  void getUserPhoto(@Path("id") int id, Callback<Photo> cb);

    • 异步模式只需要在方法的最后添加Callback，然后返回值为void

  • 配置RestAdapter的Convert

    • Retorfit默认使用Gson将Http Body转换成相应的Json
    • 可以通过实现Convert来自定义的转换器，来解析XML和其他数据

    • Example

        Gson gson = new GsonBuilder()
      .setFieldNamingPolicy(FieldNamingPolicy.LOWER_CASE_WITH_UNDERSCORES)
      .registerTypeAdapter(Date.class, new DateTypeAdapter())
      .create();

  RestAdapter restAdapter = new RestAdapter.Builder()
      .setEndpoint("https://api.github.com")
      .setConverter(new GsonConverter(gson))
      .build();

  GitHubService service = restAdapter.create(GitHubService.class);

  • 设置日志

    • Example

        RestAdapter restAdapter = new RestAdapter.Builder()
      .setLogLevel(RestAdapter.LogLevel.FULL)
      .setEndpoint("https://api.github.com")
      .build();

    • RestAdapter.LogLevel.FULL 所有的日志都会输出
